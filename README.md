# Barton Investors Group Site

Next.js App Router project for Barton Investors Group, built in JavaScript and intended for Vercel deployment with Sanity as the CMS.

## Stack

- `Next.js 16`
- `React 19`
- `Tailwind CSS 4`
- `Framer Motion`
- `Sanity`
- `next-sanity`
- `zod`

## Getting Started

New deployment allowed from parter git

## Local Mobile Testing With Ngrok

Terminal 1:

```bash
npm run dev
```

Terminal 2:

```bash
ngrok http 3000
```

Open the ngrok HTTPS URL on the phone or external device.

For absolute URLs generated by the app, set `NEXT_PUBLIC_SITE_URL` to the
public URL you want to use. In request-aware code, forwarded ngrok headers are
supported when no public base URL is configured.
